# Domain Framework: Ophthalmology Instagram Carousel Production

## Operational Process

### 1. Topic Selection & Research
- Identify topic from content calendar or trending angle
- Research medical facts from peer-reviewed sources (PubMed, AAO, CBO)
- Cross-reference at least 2 independent sources for every medical claim
- Verify CFM compliance of the topic angle before proceeding

### 2. Hook & Angle Development
- Generate 3-5 hook options using different emotional drivers (curiosity, fear, relatability, authority)
- Each hook must work within Instagram's 3-second scroll-stop window
- Test hook against: "Would I stop scrolling for this?" threshold
- Select hook that maximizes save/share potential (not just likes)

### 3. Carousel Structure Design
- Choose format: Editorial, Listicle, Tutorial, Mito vs Realidade, Problema→Solução, Storytelling
- Plan 8-10 slides with clear narrative arc
- Each slide: 40-80 words, two-layer hierarchy (headline + supporting text)
- Cover slide: bold hook + Dr. Russo/CEDOA visual identity
- Last slide: specific CTA (save, share, comment, agendar consulta)

### 4. Medical Content Adaptation
- Translate medical jargon to patient-friendly Portuguese (grade 7-8 reading level)
- "pressão intraocular elevada" → "pressão dentro do olho que pode machucar o nervo óptico"
- "ceratocone" → "quando a córnea fica fina e torta, como um cone"
- Include educational value in every slide (not just surface tips)
- Cite sources when using statistics or medical data

### 5. Caption Writing
- First 125 characters: scroll-stop hook (visible before "...mais")
- Body: expand on carousel value, use line breaks for mobile readability
- CTA: specific action ("Salve para consultar depois", "Manda para alguém que precisa")
- SEO keywords: "oftalmologista em Manaus", "cirurgia de catarata", "saúde ocular"
- 5 hashtags max: #oftalmologista #saudeocular #cedoa #manaus + 1 topic-specific

### 6. Visual Design
- Design system: consistent colors, fonts, spacing across all slides
- 1080x1440px (3:4 portrait) viewport
- Min font sizes: Hero 58px, Heading 43px, Body 34px, Caption 24px
- Alternate background colors (light/dark/accent) for visual rhythm
- Highlight key phrases in accent color
- No slide counters (Instagram shows native navigation)

### 7. Quality Review
- Medical accuracy: every claim verified against source
- CFM compliance: no guarantees, no superlatives, no patient identification
- Visual quality: fonts readable, contrast 4.5:1, no clipped content
- Copy quality: hook effectiveness, CTA presence, hashtag compliance
- Reject threshold: any criterion below 4/10 = automatic reject

### 8. Publication
- Dry-run first (validate token, upload images, create containers)
- Present preview to user (images + caption + hashtags)
- Wait for explicit user confirmation
- Publish via Instagram Graph API
- Report permalink and post ID

## Decision Criteria

### Topic Selection
- **Evergreen vs. Trending**: Lead with evergreen topics (cataract, glaucoma, dry eye) for consistent saves. Mix in trending angles (screen time, myopia in children) for reach.
- **Save-first vs. Share-first**: Checklists, symptom guides → optimized for saves. Relatable daily problems → optimized for DM shares. Alternate between both.
- **CFM Risk Assessment**: Before/after content requires 4+ patient examples showing range of outcomes. If not available, use diagram/illustration-based content instead.

### Hook Type Selection
- **Problem-Aware audience**: Use fear/urgency hooks ("5 sinais de alerta")
- **Unaware audience**: Use curiosity/relatable hooks ("Você sabia que...")
- **Solution-Aware audience**: Use authority/proof hooks ("30 anos tratando retina")

### Carousel Format Selection
- Educational explainer → Editorial or Tutorial
- Myth-busting → Mito vs Realidade
- Tips/tools/habits → Listicle
- Patient journey/results → Storytelling or Antes e Depois (with CFM compliance)
- Pain point content → Problema → Solução
